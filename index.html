Multiple Backgrounds

        The different background images are separated by commas, and the images are stacked on top of each other, where the first image is closest to the viewer.
        

            
#example1 {

            
   background-image:

            
   url(img_flwr.gif),
 
            
   url(paper.gif);

            
   background-position:
  
            
   right bottom, left top;
 
            
   background-repeat:

            
   no-repeat, repeat;
          }

        

        
CSS Tooltip

        A tooltip is used to specify more information when the mouse is pointed over the element.
        

            
.tooltip .tooltiptext { 

            
   visibility: hidden;

            
    width: 120px;

            
    background-color: black;

            
    color: #fff;

            
    text-align: center;

            
    padding: 5px 0;
 
            
    border-radius: 6px;

            
  }
